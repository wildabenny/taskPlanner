TaskPlanner
===========

Funkcjonalności:
- tworzenie użytkowników
- logowanie, dostęp dla określonych grup
- dodawanie zadań
- wyświetlanie zadań, możliwość zobaczenia szczegółów
- walidacja wprowadzanych danych
- dodawanie kategorii do zadań
- edycja zadań

 Użyte technologie: <br>
 - Symfony 2
 - Doctrine
 - MySQL
 - JQuery
 - AJAX
 - FOSUserBundle
 - JsRoutingBundle
 - Bootstrap
